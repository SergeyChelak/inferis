# Inferis
Doom-style pseudo 3D first person shooter game

## Prerequisites
### Cargo/Rust language
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### SDL2
OS X
```
brew install sdl2 sdl2_image
```
Arch
```
sudo pacman -S sdl2 sdl2_image
```
